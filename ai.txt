# Description
Description: AI Nexus is a curated directory of AI tools (2000+ verified entries) with categories like Chat, Image, Video, Coding, Writing, Audio, and Productivity. The tools.json endpoint provides a featured subset of 20-50 popular tools with complete metadata.

# Usage
Access: read-only
Rate-limit: Please avoid abusive scraping. For bulk data access, use the tools.json endpoint.

# Schema
Tools follow this schema: name, slug, url, category, tags[], description, pricing, icon, verified, lastUpdated
